Design Dev Dive is a personal project built to explore both front-end development and user experience principles through the lens of modern design tools. The site showcases the popular platform of Figma, with a focus on features that often go unnoticedâ€”but can significantly improve design workflows.

This project was developed for a web development course and serves as a portfolio piece highlighting modular JavaScript, responsive design, and accessible interface structure. Each tool profile is built from dynamic data, incorporating ES modules, arrays, and component-based UI logic. At the same time, care has been taken to create a clean, inviting user experience that reflects thoughtful UX design.

Frame Tool:
Used to create frames, which are the foundation of your designs and act as containers for other elements.

Shape Tools: Includes rectangles, lines, arrows, ellipses, polygons, and stars for building the visual structure of your designs.

Pen Tool: Allows you to draw custom shapes and icons by creating paths and curves.

Text Tool: Enables the addition of text layers to your designs, with options for styling and formatting.

Move Tool: Used to select and manipulate layers on the canvas, including resizing and positioning.

Color Picker: Allows you to select colors and adjust opacity for various design elements.

Gradients: Provides options for creating and applying gradients to fill shapes and other elements.

Styles: Allows you to define and reuse styles for colors, text, effects, and layout guides, ensuring consistency across your designs.

Components: Enables the creation of reusable design elements that can be updated globally, ensuring consistency and efficiency.
